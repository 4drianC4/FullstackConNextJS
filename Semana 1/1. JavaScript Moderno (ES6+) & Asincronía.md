> [!INFO] Objetivo de la clase
>Actualizar el "chip" mental. En React no usamos bucles `for`, ni modificamos variables directamente, ni usamos `function` antigua. Hoy aprenderemos la sintaxis que usarás el 90% del tiempo en el curso.
# 1. El cambio de mentalidad
En el desarrollo moderno, no le decimos al navegador *cómo* hacer las cosas paso a paso (Imperativo), sino *qué* queremos mostrar (Declarativo). Para eso, necesitamos herramientas de JS que manipulen datos, no HTML.
# 2. Desestructuración (Destructuring)
Extraer datos de objetos y arreglos sin repetir código.
## Antes (Código sucio)
```javascript
const usuario = { nombre: "Alex", edad: 25 };
const nombre = usuario.nombre;
const edad = usuario.edad;
console.log(nombre);
```
## Ahora (React Style)
```JavaScript
const usuario = { nombre: "Alex", edad: 25, rol: "Admin" };

// Extraemos solo lo que necesitamos
const { nombre, rol } = usuario; 

console.log(nombre); // "Alex"
```
>[!TIP] ¿Por qué importa en React? 
>Lo usaremos todo el tiempo para recibir las **Props** en los componentes: `function Tarjeta({ titulo, imagen }) { ... }`
