Para asegurar que cada parte funcione correctamente y no abrumarnos con configuraciones, dividiremos la creación en estas 4 fases:

1. **Frontend y Estructura Base:** Configuración de Next.js (que ya incluye React bajo el capó), TypeScript y Tailwind CSS.
    
2. **Base de Datos:** Levantamiento de un contenedor de PostgreSQL usando Docker.
    
3. **Backend y ORM:** Instalación y configuración de Prisma para interactuar con la base de datos desde nuestras rutas de Next.js.
    
4. **Control de Versiones:** Inicialización del repositorio y subida a GitHub.
