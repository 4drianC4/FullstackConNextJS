Para empezar lo primero que se debe hacer es revisar si se tiene instalado node y npm, como se muestra en [[CLASE 1. SEMANA 1#Fundamentos, Entorno y Flujo de Trabajo Colaborativo#Node.js y npm]], luego de esto podremos continuar con los pasos que siguen
# Pasos para instalar
Vamos a iniciar el proyecto usando la herramienta oficial de Next.js. Esta herramienta configurará automáticamente React, TypeScript y Tailwind CSS por nosotros en un solo paso. Primeramente abre una terminal y dirígete a la carpeta donde quieras tener el proyecto, desde aquí debes escribir el siguiente comando:
```Bash
npx create-next-app@latest nombre-proyecto
```

se te preguntará:

? Would you like to use the recommended Next.js defaults? › - Use arrow-keys. Return to submit.
❯   Yes, use recommended defaults - TypeScript, ESLint, Tailwind CSS, App Router
    No, reuse previous settings
    No, customize settings
    
elegimos la primera opción y así tendremos todo lo necesario automáticamente. Tendremos algo similar a lo siguiente:
![[Pasted image 20260312110749.png|1000]]

una vez completada la instalación local, ingresamos a la carpeta creada y pasaremos a correr el proyecto con los siguientes comandos:
```Bash
cd mi-proyecto
npm run dev
```
> [!INFO] para parar el proyecto presiona ctrl + c
